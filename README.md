| 7>5 olduğu için *5 i 7 nin soluna yazıyoruz |
| 5>1 olduğu için *1 i 5 in  soluna yazıyoruz |
| 8>7 olduğu için *8 i 7 nin sağına yazıyoruz |
| 3>1 olduğu için *3 ü 1 in  soluna yazıyoruz |
